# 🚀 Simple Python-to-SQL ETL Pipeline

## Overview
This project extracts employee data from a CSV, cleans it using Pandas, and loads it into a SQLite database.

## Tech Stack
- **Language:** Python 3.14
- **Libraries:** Pandas, SQLite3
- **Tools:** VS Code, Git

## How to Run
1. Activate venv: `.\venv\Scripts\activate`
2. Install dependencies: `pip install pandas`
3. Run script: `python main.py`
